# 📋 Информация о проекте

## Что это?

Telegram Mini App - многопользовательская игра "Угадай слово" (аналог Wordle на русском языке).

## Создано: 09.05.2026

## Технический стек

### Backend
- **Node.js** - серверная платформа
- **Express** - веб-фреймворк
- **node-telegram-bot-api** - интеграция с Telegram
- **SQLite3** - база данных
- **dotenv** - управление переменными окружения
- **cors** - поддержка CORS

### Frontend
- **Vanilla JavaScript** - без фреймворков
- **HTML5** - разметка
- **CSS3** - стили и анимации
- **Telegram Web App SDK** - интеграция с Telegram

### База данных (SQLite)

**Таблица users:**
- telegram_id (PRIMARY KEY)
- username
- first_name
- games_played
- games_won
- total_score
- created_at

**Таблица daily_games:**
- id (AUTO INCREMENT)
- telegram_id (FOREIGN KEY)
- date
- word
- attempts
- won
- score
- time_taken
- created_at

**Таблица daily_words:**
- date (PRIMARY KEY)
- word

## Основные функции

### Игровая механика
- ✅ Угадывание слова из 5 букв за 6 попыток
- ✅ Цветовая индикация (зеленый/желтый/серый)
- ✅ Виртуальная клавиатура на русском
- ✅ Поддержка физической клавиатуры
- ✅ Таймер игры
- ✅ Счетчик попыток

### Система очков
- Базовые очки: 100
- Бонус за попытки: (6 - attempts) × 20
- Бонус за скорость: до 50 очков
- Максимум: 250 очков

### Многопользовательские функции
- ✅ Таблица лидеров (топ-10)
- ✅ Статистика игрока
- ✅ Ежедневное слово для всех
- ✅ Защита от повторной игры в один день

### Telegram интеграция
- ✅ Telegram Bot с командами
- ✅ Mini App интерфейс
- ✅ Кнопка "Поделиться результатом"
- ✅ Автоматическая авторизация через Telegram

## API Endpoints

### User Management
- `POST /api/user/init` - Инициализация пользователя
- `GET /api/user/:telegram_id` - Получение данных пользователя

### Game Logic
- `GET /api/daily-word` - Получение слова дня
- `POST /api/game/complete` - Сохранение результата игры
- `GET /api/today-played/:telegram_id` - Проверка игры за сегодня

### Leaderboard
- `GET /api/leaderboard` - Топ-10 игроков

## Telegram Bot команды

- `/start` - Начать игру (показывает кнопку Mini App)
- `/stats` - Показать статистику игрока
- `/leaderboard` - Показать таблицу лидеров
