# 🎮 Telegram Word Game

Многопользовательская игра "Угадай слово" для Telegram Mini Apps. Угадывайте слова из 5 букв за 6 попыток, соревнуйтесь с друзьями и поднимайтесь в таблице лидеров!

## 🎯 Особенности

- 🔤 Ежедневное слово для всех игроков
- 🎨 Красивый интерфейс с анимациями
- 🏆 Таблица лидеров
- 📊 Статистика игр
- ⏱️ Система очков с бонусами за скорость
- 🇷🇺 Полностью на русском языке
- 📱 Адаптивный дизайн для мобильных устройств

## 🚀 Быстрый старт

Подробная инструкция по установке и настройке находится в файле [SETUP.md](SETUP.md).

### Краткая версия:

```bash
# 1. Установите зависимости
npm install

# 2. Создайте .env файл
cp .env.example .env

# 3. Добавьте токен бота в .env
# BOT_TOKEN=ваш_токен_от_botfather

# 4. Запустите сервер
npm start

# 5. Запустите ngrok для публичного URL
ngrok http 3000
```

## 🎮 Как играть

1. Откройте бота в Telegram
2. Нажмите `/start` и кликните "🎮 Играть"
3. Угадайте слово из 5 букв за 6 попыток
4. После каждой попытки буквы окрашиваются:
   - 🟩 **Зеленый** - буква на правильном месте
   - 🟨 **Желтый** - буква есть в слове, но не на этом месте
   - ⬜ **Серый** - буквы нет в слове

## 🏆 Система очков

- Базовые очки: **100**
- Бонус за попытки: **(6 - количество попыток) × 20**
- Бонус за скорость: **до 50 очков**

Максимум: **250 очков** (угадать с первой попытки за 0 секунд)

## 🤖 Команды бота

- `/start` - Начать игру
- `/stats` - Показать вашу статистику
- `/leaderboard` - Таблица лидеров

## 🛠️ Технологии

- **Backend**: Node.js, Express, node-telegram-bot-api
- **Database**: SQLite3
- **Frontend**: Vanilla JavaScript, HTML5, CSS3
- **API**: Telegram Mini Apps SDK
