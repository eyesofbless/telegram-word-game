# ✅ Чеклист запуска

Используйте этот чеклист для запуска вашей игры в Telegram.

## Подготовка (5 минут)

- [ ] Node.js установлен (проверьте: `node --version`)
- [ ] npm установлен (проверьте: `npm --version`)
- [ ] Проект скачан в `C:\web\what_if`

## Шаг 1: Создание Telegram бота (2 минуты)

- [ ] Открыли [@BotFather](https://t.me/BotFather) в Telegram
- [ ] Отправили команду `/newbot`
- [ ] Ввели имя бота (например: "Word Game Bot")
- [ ] Ввели username бота (например: "my_word_game_bot")
- [ ] **Скопировали токен бота** (сохраните его!)

## Шаг 2: Настройка проекта (1 минута)

- [ ] Открыли папку `C:\web\what_if` в терминале
- [ ] Выполнили: `npm install`
- [ ] Скопировали `.env.example` в `.env`
- [ ] Вставили токен бота в `.env` файл:
  ```
  BOT_TOKEN=ваш_токен_здесь
  WEB_APP_URL=http://localhost:3000
  PORT=3000
  ```

## Шаг 3: Локальный запуск (30 секунд)

- [ ] Выполнили: `npm start` (или `start.bat` на Windows)
- [ ] Сервер запустился на http://localhost:3000
- [ ] Открыли http://localhost:3000 в браузере
- [ ] Игра работает локально ✅

## Шаг 4: Публичный URL с ngrok (2 минуты)

- [ ] Установили ngrok: `npm install -g ngrok`
- [ ] Запустили в новом терминале: `ngrok http 3000`
- [ ] Скопировали HTTPS URL (например: `https://abc123.ngrok-free.app`)
- [ ] Обновили `.env`:
  ```
  WEB_APP_URL=https://abc123.ngrok-free.app
  ```
- [ ] Перезапустили сервер (`Ctrl+C` и снова `npm start`)

## Шаг 5: Подключение Mini App (3 минуты)

- [ ] Открыли [@BotFather](https://t.me/BotFather)
- [ ] Отправили команду `/newapp`
- [ ] Выбрали вашего бота из списка
- [ ] Ввели данные:
  - **Title**: Word Game
  - **Description**: Угадай слово за 6 попыток!
  - **Photo**: [Send /empty чтобы пропустить]
  - **Demo GIF/Video**: [Send /empty чтобы пропустить]
  - **Web App URL**: `https://abc123.ngrok-free.app` (ваш ngrok URL)
  - **Short name**: `wordgame` (или другое уникальное имя)
- [ ] Получили подтверждение от BotFather ✅

## Шаг 6: Тестирование (1 минута)

- [ ] Нашли вашего бота в Telegram (по username)
- [ ] Отправили `/start`
- [ ] Увидели приветственное сообщение
- [ ] Нажали кнопку "🎮 Играть"
- [ ] Mini App открылось ✅
- [ ] Игра работает! 🎉

## Проблемы и решения

### ❌ "Bot token is invalid"
- Проверьте правильность токена в `.env`
- Убедитесь что нет лишних пробелов
- Перезапустите сервер

### ❌ Mini App не открывается
- Убедитесь что ngrok запущен
- Проверьте что URL в `.env` совпадает с ngrok URL
- URL должен быть HTTPS (не HTTP)
- Перезапустите сервер после изменения `.env`

### ❌ "Cannot find module"
- Выполните: `npm install`
- Проверьте что вы в правильной папке

### ❌ База данных не создается
- Проверьте права на запись в папке проекта
- Попробуйте запустить от имени администратора

### ❌ ngrok показывает ошибку
- Бесплатная версия ngrok может требовать регистрацию
- Зарегистрируйтесь на https://ngrok.com
- Выполните: `ngrok authtoken ваш_токен`
