🚀 БЫСТРЫЙ СТАРТ

1. cd C:\web\what_if
2. npm install
3. Создайте бота в @BotFather
4. Скопируйте .env.example в .env
5. Добавьте BOT_TOKEN в .env
6. npm start
7. Откройте http://localhost:3000

Для Telegram Mini App:
8. ngrok http 3000
9. Обновите WEB_APP_URL в .env
10. Настройте Mini App в @BotFather

Подробнее: см. QUICKSTART.md или CHECKLIST.md
